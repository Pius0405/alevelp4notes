Temrs and Definitions

Term Definition
Class A template defining the methods and data of a certain type of object
Object An instance of a class
Instance An occurence of an object
Encapsulation Putting data and methods together as a single unit (a class)
Attributes The data items of a class
Methods The subroutines of a class acting on the attributes
Getter A method used to return the value of a property
Setter A method used to update the value of a property
Inheritance The capability of defining a new derived class that has all attributes and methods from the parent class
Polymorphism Methods can be redefined for derived classes
Containment A relationship in where a class has a component that is of the other class type

Inheritance

Example code:

class Employee:

  def __init__(self,name,age,salary):

    self.__name = name

    self.__age = age

    self.__salary = salary

  def ShowDetails(self):

    print(self.__name)

    print(self.__age)

    print(self.__salary)

class Manager(Employee):

  def __init__(self,name,age,salary,managerOffice):

    Employee.__init__(self,name,age,salary)

    self.__managerOffice = managerOffice

  def showDetails(self):

    Employee.ShowDetails(self)

    print(self.__managerOffice)


Employee is the parent class while Manager is the derived class.

Derived class Manager has all attributes and methods from parent class Employee

Containment

Example code:

class Course:

  def __init__(self, t, m):

    self.__CourseTitle = t

    self.__MaxStudents = m

    self.__NumberOfLessons = 0

    self.__CourseLesson = []

  def AddLesson(self, t, d, r):

    self.__NumberOfLessons = self.__NumberOfLessons + 1

    self.__CourseLesson.append(Lesson(t, d, r))

  def OutputCourseDetails(self):

    print(self.__CourseTitle, "Maximum number: ", self.__MaxStudents)

    for i in range(self.__NumberOfLessons):

      print('Lesson',str(i+1))

      print(self.__CourseLesson[i].OutputLessonDetails())


class Lesson:

  def __init__(self,t,d,r):

    self.__LessonTitle = t

    self.__Duration = d

    self.__RequiresLab = r

  def OutputLessonDetails(self):

    print('Title:',self.__LessonTitle)

    print('Duration',self.__Duration)

    if self.__RequiresLab:

      print('Requires lab')

    else:

      print('No lab required')
