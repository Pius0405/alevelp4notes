File modes

  1. 'r'

    Opens a file for reading only

  2. 'w'

    Opens a file for writting only

    If the file exists it overwrites the whole file, otherwise it creates a new file for writting

  3. 'a'

    Opens a file for appending at the end of the file If the file does not exists it creates a new file

e.g. thisFile = open('file.txt','r')


Read operations

  1. readline()

    Reads a single line of the file

  2. read()

    Reads the whole content of the file

  3. readlines()

    Returns a list of all the lines read from the file


Write operation

  1. write()

    directly write a string which is passed as an argument to a file

  2. writelines()

    write multiple lines into a specific file, with a list of string elements passed as an argument

Always close a file after opening it!

e.g. thisFile.close()


Random file

  1. Library used : pickle

    import pickle

  2. Write a whole record to the file

    pickle.dump(<record>,<filename>)

  3. Read a record from the file

    pickle.load(<filename>)

  4. Move file pointer to point to a location

    file.seek(<address>)

  5. Use a hashing function to calculate an address from the record key

    e.g. hash(thisCar.vehicleID)
